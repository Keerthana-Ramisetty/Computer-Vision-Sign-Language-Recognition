# Computer Vision Sign Language Recognition

Developed an AI-powered Sign Language Recognition System using Python, Computer Vision, OpenCV, Machine Learning, and Deep Learning for real-time hand gesture detection and classification.

## Key Features
- Real-time hand gesture recognition and gesture-to-text conversion
- Image Processing, Feature Extraction, and Gesture Classification
- Machine Learning and Deep Learning based predictive modeling
- Model Evaluation and Performance Optimization

## Technologies Used
- Python
- OpenCV
- NumPy
- Machine Learning
- Deep Learning
- Computer Vision
- Data Preprocessing
- Feature Engineering

## Skills Demonstrated
Computer Vision, Deep Learning, OpenCV, Image Processing, Feature Extraction, Gesture Recognition, Classification, Model Evaluation, Python, Machine Learning
